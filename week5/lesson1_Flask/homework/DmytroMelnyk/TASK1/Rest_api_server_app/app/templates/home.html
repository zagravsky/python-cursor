Welcome to Cars REST API Server

GET api/cars --------------------------------> get a full list of cars in Carbase

GET api/cars/car_model ----------------------> get the car_model of cars in Carbase

For Add/Remove cars to/from carbase you need Insomnia or Postman

POST api/cars/ ----------------------> add the car_model of car in Carbase

DEL api/cars/car_model ----------------------> get the car_model of cars in Carbase
